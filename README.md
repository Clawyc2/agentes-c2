# Agentes C2

Dashboard de gestión de subagentes para Clawy.

## 🐾 ¿Qué es?

Sistema de monitoreo y gestión de subagentes donde Clawy (líder) coordina equipos de trabajo especializados.

## 🚀 Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Clawyc2/agentes-c2)

## 📊 Features

- 👑 Jerarquía de agentes (Líder, Senior, Mid, Junior, Especialista)
- 📋 Gestión de tareas y proyectos
- 💬 Visualización de conversaciones
- 📈 Estadísticas de uso y rendimiento
- ✏️ Edición de reglas y comportamientos
- 🗑️ Eliminación de agentes

## 🛠️ Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
- Framer Motion
- Lucide Icons
